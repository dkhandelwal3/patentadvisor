## patent-service 
This is for backend services

### How to run 

1. Add .env file

   Add the .env file in patent-service folder and add the below properties in .env file. Replace SETUP_PATH with the actual path. 

     HOME_PATH=${SETUP_PATH}/patentadvisor/patent-service
     OPENAI_API_KEY=""

2. Run main.py
